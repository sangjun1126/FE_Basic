## 9장 : 링크 만들기

* 링크란 현재 문서에서 다른 문서로 이동할 수 있는 수단을 의미합니다.

---

### 링크는 anchor

* a 태그 요소는 href 속성을 통해 다른 페이지, 전화번호, 이메일 주소와 그 외 다른 url로 연결이 가능한 링크(연결)을 만듭니다.

* 인라인 요소이며, 콘텐츠는 주로 링크의 목적지를 나타냅니다.

```javascript
<a href = 'https://www.naver.com'>네이버 링크로 들어가기</a>
```

---

### 새 탭에서 열고 싶을 경우

* a 태그의 target 속성을 이용하면 새로운 문서를 열 경우 현재 탭에서 열지, 새로운 탭에서 열지 결정할 수 있습니다.

```javascript
<a href = "https://www.naver.com" target = "_self">현재 탭에서 열기 (기본값)</a>

<a href = "https://www.naver.com" target = "_blank">새 탭에서 열기</a>
```

---

### 연략을 해보자

* a 태그의 href에는 웹 문서의 주소 뿐 아니라 전화번호나 메일 주소 등을 저장할 수도 있습니다. 이때 유형별로 추가되는 텍스트가 있습니다.

```javascript
<a href = "tel:010-1234-5678" target="_self">전화걸기</a>

<a href = "jeongsangjun@naver.com" target="_blank">메일 쓰기</a>
```

---

### 내용 정리

* a 태그는 페이지를 다른 위치로 이동시킬 수 있는 링크를 만듭니다.

* href 속성에는 링크가 이동하고자 하는 url를 입력합니다.

* target 속성에는 새 페이지가 열릴 대상 즉 탭을 지정할 수 있습니다.

* href 속성을 통해 전화 걸기, 이메일 보내기 등을 지정할 수도 있습니다.
